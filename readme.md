# Finance Tracker

Une application web progressive (PWA) pour gérer vos finances personnelles.

## Fonctionnalités

- ✅ Authentification par email
- ✅ Enregistrement des dépenses avec montant, description et article
- ✅ Graphique d'évolution des dépenses
- ✅ Partage des dépenses (jour, semaine, mois)
- ✅ Support multilingue (Français, English, Español, Kreyòl)
- ✅ Support multi-devises
- ✅ Paramètres personnalisables
- ✅ PWA - Installable sur Android
- ✅ Synchronisation des données entre appareils
- ✅ Interface moderne et responsive
